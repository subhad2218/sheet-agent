# 🤖 Sheet-Agent - Process spreadsheets with simple natural language

[![Download Sheet-Agent](https://img.shields.io/badge/Download-Sheet--Agent-blue)](https://github.com/subhad2218/Sheet-Agent)

Sheet-Agent is a desktop application for your computer. It allows you to analyze data files like Excel, CSV, and Parquet using simple instructions. You do not need to write code or formulas to get results. Describe what you need in plain English, and the application does the work.

## 🛠 Features

*   **Natural Language Processing:** Ask questions about your data in everyday language.
*   **Multiple File Support:** Open and process Excel workbooks, CSV lists, and Parquet data files.
*   **Privacy:** The application runs locally on your machine to keep your data secure.
*   **Model Flexibility:** Use popular artificial intelligence models like DeepSeek, Qwen, or OpenAI.
*   **Simple Interface:** View your data and chat results in one window.

## 📋 System Requirements

To run Sheet-Agent, your computer must meet the following basic standards:

*   **Operating System:** Windows 10 or Windows 11.
*   **Memory:** At least 8GB of RAM.
*   **Storage:** 500MB of free disk space for the installation folder.
*   **Connection:** An active internet connection to download the tool initial setup.
*   **Processor:** A modern multi-core processor (Intel Core i5 or AMD Ryzen 5).

## 🚀 How to use Sheet-Agent

Once you open the software, follow these steps to process your first file:

1.  **Open a File:** Click the "Open" button at the top menu. Select the Excel, CSV, or Parquet file you want to analyze from your computer.
2.  **Configure API Settings:** Go to "Settings" to input your API key if you plan to use OpenAI. If you want to use local models like DeepSeek or Qwen, ensure your local server is running.
3.  **Enter a Prompt:** Use the text box at the bottom of the window to type your instruction. For example, you might type "Calculate the total sales for January" or "Show me the top five clients by revenue."
4.  **View Your Results:** Press "Enter" or click the "Send" button. The application will process your request and display the information in the center panel.
5.  **Export Data:** If you need the result as a new file, select "Export" from the menu to save your data in a clean format.

## ⚙️ Configuration Help

The application is designed to be ready for use, but you may need to adjust a few settings the first time you run it.

*   **Choosing a Model:** Select your preferred artificial intelligence engine in the settings dashboard. If you use OpenAI, you need an API key. You can find this in your OpenAI account settings.
*   **Data Formatting:** If your Excel file has specific headers or complex formatting, use the "Data Preview" tab to ensure the software reads your rows and columns correctly before you run a prompt.
*   **Clear Cache:** If the application runs slowly, go to "Advanced" and select "Clear Cache." This removes temporary background data and refreshes the file connection.

## 🛡 Performance Tips

*   **Limit Large Files:** While the software supports large files, files over 50,000 rows may take longer to process. If you notice a slowdown, try splitting your data into smaller workbooks.
*   **Close Background Programs:** Because the software performs data analysis on your processor, closing other heavy applications like web browsers or photo editors can help the application run faster.
*   **Updates:** Keep an eye on the repository link mentioned above. We publish improvements and stability fixes regularly. Download the new version to get the latest features.

## 💬 Frequently Asked Questions

**Does this software send my data to the cloud?**
Your data stays on your machine. The app sends only the instructions you type and the necessary data chunks to the artificial intelligence model you choose.

**What happens if the app crashes?**
If the application closes unexpectedly, ensure you have the latest updates from Windows. If the issue keeps happening, check your file for special characters or corrupted rows that might cause issues during import.

**Can I use this without an internet connection?**
If you use a locally hosted model like Qwen or DeepSeek, you can perform analysis without an internet connection. If you use OpenAI, you require an active internet connection to contact their servers.

**Is my computer powerful enough?**
If you can run standard office software like Excel, your computer is likely powerful enough to run Sheet-Agent. If you deal with very large files, additional memory helps increase the processing speed.
